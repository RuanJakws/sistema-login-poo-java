# sistema-login-poo-java

Projeto simples de sistema de login desenvolvido em Java utilizando Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades

- Cadastro de usuários
- Login com validação de email e senha
- Armazenamento em memória utilizando ArrayList
- Estrutura orientada a objetos com classe Usuario

## 🛠️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
